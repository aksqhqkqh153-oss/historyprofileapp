# HistoryProfile 마이크로 인플루언서 플랫폼 방향 정리

## 1. 서비스 개요
- 프로필 기반 마이크로 인플루언서 플랫폼
- 사용자 = 광고 매체 (Profile, Q&A, Feed)

## 2. 수익 구조
### 2.1 광고 수익
- Google AdSense (질문/답변 페이지)
- 홈 피드 네이티브 광고
- 직접 광고 (스폰서드 카드)
- 키워드 경쟁 입찰

### 2.2 핵심 수익 공식
- 플랫폼 수익 = 광고수익 + 직접광고 + 키워드입찰 - 사용자 리워드

## 3. 사용자 활동 → 수익 흐름
- 사용자 활동
- 콘텐츠 생성 (프로필/질문/답변)
- 트래픽 발생
- 광고 노출
- 플랫폼 수익 발생
- 일부 포인트 환원

## 4. 포인트 지급 로직
### 4.1 기본 지급 기준
- 질문 받기: +1P
- 답변 작성: +3P
- 프로필 조회 100회: +5P
- DM 유입: +2P

### 4.2 보너스 지급
- 인기 답변: +10P
- 키워드 상위 노출 유지: +5P/day

### 4.3 금지 사항
- 광고 클릭 보상 금지
- 광고 시청 보상 금지

## 5. 예상 수익 계산 로직
- 활동 점수 = (질문 + 답변 + 조회수 + DM)
- 예상 포인트 = 활동점수 × 가중치
- 예상 수익 = 포인트 × 환산 비율

## 6. 출금 정책
- 최소 출금: 10,000P
- 출금 주기: 월 1회
- 관리자 승인 필수

## 7. 키워드 경쟁 구조
- 키워드별 포인트 입찰
- 상위 노출 = 더 많은 질문/DM 유입
- 경쟁 UI 제공

## 8. 직접 광고 구조
- 사용자/기업 광고 신청
- 관리자 승인
- 피드 중간 노출

## 9. 관리자 시스템
- 출금 승인/반려
- 광고 승인
- 키워드 경쟁 관리

## 10. 리스크 대응
- AdSense 정책 준수
- 부정 활동 감지
- 포인트 남용 방지

## 11. 성장 전략
- 프로필 → 질문 → 광고 → 수익 구조 강화
- 유저를 광고 매체화
- B2B 광고 확장

## 핵심 문장
> 유저는 콘텐츠를 만들고, 플랫폼은 수익을 만든다
