# 마이크로 인플루언서 플랫폼 운영 메모

## 서비스 방향
- 메인 자산: 프로필, 경력, 자기소개, 프로필연동, 파일관리(자기소개서/경력기록/증빙/자격증)
- 전환 구조: 공개 프로필 → 질문/답변/DM → 광고 노출 및 문의 전환
- 보상 구조: 광고 시청 보상 금지, 활동 기반 포인트 적립, 최소 10,000P부터 월 1회 출금

## 비교 포지션
- LinkedIn: 전문성은 강하지만 보상 구조가 약함
- Instagram: 노출은 강하지만 경력/증빙 자산 관리가 약함
- 크몽/숨고형: 문의 전환은 빠르지만 장기 브랜딩 자산 축적이 약함
- 본 앱: 프로필 자산 관리 + 질문/DM 전환 + 광고 운영 + 포인트 리워드 통합

## 운영 원칙
1. 광고주는 플랫폼이 모집하고 관리한다.
2. 회원은 자신의 프로필/피드/질문 활동으로 노출을 만든다.
3. 리워드는 광고 직접 시청이 아니라 활동 점수 기준으로 환산한다.
4. 키워드 경쟁과 직접 광고는 포인트 사용형으로 운영한다.
